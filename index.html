使い方ガイド

📝 基本操作

  • 実行: コード全体を一度に実行します
  • 1行実行: コードを1行ずつ実行し、変数の変化を確認できます
  • リセット: 実行状態をリセットします

🔢 使える演算子

  • + 足し算 (例: 5 + 3 = 8)
  • - 引き算 (例: 10 - 4 = 6)
  • * 掛け算 (例: 3 * 4 = 12)
  • / 割り算 (例: 10 / 2 = 5)
  • % 余り (例: 10 % 3 = 1)
  • == 等しい (例: 5 == 5 は真)
  • != 等しくない (例: 5 != 3 は真)
  • < より小さい
  • > より大きい
  • <= 以下
  • >= 以上

📦 使える機能

  • print(値) - 値を表示
  • 変数 = 値 - 変数に値を代入
  • if 条件: - 条件分岐
  • while 条件: - 繰り返し
  • for 変数 in リスト: - リストの各要素を処理
  • [1, 2, 3] - リスト（配列）
  • リスト[0] - リストの要素にアクセス

💡 コード例

# 変数と計算
a = 10
b = 3
print(a + b)    # 13
print(a % b)    # 1 (余り)

# 条件分岐
if a > b:
    print("aの方が大きい")

# リスト
numbers = [1, 2, 3]
print(numbers[0])  # 1

# 繰り返し
for n in numbers:
    print(n * 2)

⚠️ 注意事項

  • 0で割ることはできません
  • インデント（字下げ）は4つのスペースを使用してください
  • 文字列は " または ' で囲みます
