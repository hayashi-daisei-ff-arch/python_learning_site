使い方ガイド

📝 基本操作

  • 実行: コード全体を一度に実行
  • 1行実行: 1行ずつ実行し変数の変化を確認
  • リセット: 実行状態をリセット
  • コード追加: フォームで簡単にコード生成

🔢 使える演算子

  • + 足し算, - 引き算, * 掛け算
  • / 割り算, % 余り
  • == 等しい, != 等しくない
  • <, >, <=, >= 比較

🔄 型変換関数

  • int(値) - 整数に変換 (例: int(3.14) → 3)
  • float(値) - 小数に変換 (例: float(5) → 5.0)
  • str(値) - 文字列に変換 (例: str(123) → "123")

📦 使える機能

  • print(値) - 値を表示
  • 変数 = 値 - 変数に代入
  • if 条件: - 条件分岐
  • while 条件: - 繰り返し
  • for 変数 in リスト: - リスト処理
  • [1, 2, 3] - リスト, リスト[0] - 要素アクセス

💡 コード例

# 型変換
x = 3.14
y = int(x)      # 3 (整数)
z = str(y)      # "3" (文字列)
print(z)

# 計算と余り
print(10 % 3)   # 1

# リスト
nums = [1, 2, 3]
for n in nums:
    print(n * 2)

⚠️ 注意事項

  • 0で割ることはできません
  • インデントは4つのスペース
  • 文字列は " または ' で囲む
